# /salute-speech -- Работа с речью через SaluteSpeech

## Используемые инструменты

| Инструмент | Что делает |
|------------|------------|
| `recognize_speech` | Распознавание речи из аудио в текст |
| `synthesize_speech` | Синтез речи из текста в аудио |

## Алгоритм

1. Определи тип задачи: распознавание или синтез
2. Для распознавания: прочитай аудиофайл, закодируй в Base64, вызови `recognize_speech`
3. Для синтеза: вызови `synthesize_speech` с текстом и голосом
4. Покажи результат

## Формат ответа

### Распознавание
```
## Результат распознавания

**Текст**: <распознанный текст>
**Язык**: ru-RU
```

### Синтез
```
## Результат синтеза

**Голос**: Nec_24000
**Формат**: opus
**Размер**: 12345 байт
```

## Примеры

```
/salute-speech synthesize Привет, как дела?
/salute-speech recognize audio.wav
```
